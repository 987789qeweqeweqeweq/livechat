# Citi Live Chat Support

Secure, regional live chat support page for Citi customers.

## Getting Started

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

Copy `.env.example` to `.env.local` and configure platform-specific tool URLs. All variables are optional — unset platforms fall back to AnyDesk.

## Build

```bash
pnpm build
pnpm start
```
